# 🎓 EduJob: AI-Based Job Prediction System

## 📌 Project Overview
EduJob is a Machine Learning-based web application that predicts suitable job roles for students based on their academic performance, skills, and experience.  

The system uses a **Random Forest Classifier** to analyze user inputs and suggest the top 3 most suitable job roles.

---

## 🎯 Objectives
- Help students identify suitable career paths  
- Use AI to analyze skills and academic data  
- Provide accurate job predictions  
- Build an interactive web application using Streamlit  

---

## 🚀 Features

### 🔐 User Authentication
- User Registration and Login system  
- Email validation (only Gmail allowed)  
- Strong password validation  

### 🧾 Student Details Input
- Degree and Specialization  
- Programming Skills  
- Internship Experience  
- CGPA  
- Projects completed  
- Certifications  
- Problem-solving ability  

### 🤖 AI Job Prediction
- Uses **Random Forest Algorithm**  
- Predicts **Top 3 job roles**  
- Displays probability percentage  
- Shows best match  

### 📊 Data Visualization
- Job role distribution graph  
- Programming skills demand graph  

### 📜 History Tracking
- Stores previous predictions  
- Displays user-wise history  

---

## 🛠 Technologies Used

| Category | Tools |
|---------|------|
| Frontend | Streamlit |
| Backend | Python |
| ML Model | Random Forest |
| Libraries | Pandas, Scikit-learn, Matplotlib |
| Data Storage | CSV Files |

---

## 🧠 Machine Learning Model

- Algorithm: **Random Forest Classifier**
- Data preprocessing:
  - Label Encoding (for target)
  - One-hot encoding (for features)
- Train-Test Split: 80% training, 20% testing
- Evaluation Metrics:
  - Training Accuracy
  - Testing Accuracy
